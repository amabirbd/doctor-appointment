# Doctor appointment MERN app

## Usage

make sure you have installed and running mongodb in your system.

Rename the .envexample to .env

You can use the default MONGO_URI or you can add your own.

### Install dependencies

```
# Backend deps
at root folder
npm install

# Frontend deps
cd frontend
npm install
```

### Run App

At root folder run -

```
npm run dev
```
